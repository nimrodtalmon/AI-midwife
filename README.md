# AI-Midwife

A domain-general tool for crystallizing what you want to make or do. It elicits your *will* through a rapid stream of tappable multiple-choice questions, accumulates your answers as atomic commitments, and renders an artifact that reflects the full shape of your intent — maximizing will extracted per unit cognitive effort.

## How to run

1. Open `index.html` in any modern browser (no build step needed), or deploy the folder to GitHub Pages.
2. Paste an [Anthropic API key](https://console.anthropic.com/) when prompted — it's stored in `localStorage` and never leaves your browser.
3. Tap through the questions. Use **Free-text** to add detail, **Pass** to get fresh options for the same question, **Skip** to move to a different question.
4. Click **Show draft** at any time to see the current artifact rendered from your commitments.
5. Click any commitment in the sidebar to retract it.
6. **Reset** clears all state and restarts from the cold-start question.
